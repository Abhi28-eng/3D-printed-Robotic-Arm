# 3D-printed-Robotic-Arm
Wireless 3D-printed robotic arm using Raspberry Pi Pico + PCA9685 with 7 servos (MG996R + MG90S). Controlled through a custom ESP32 + NRF24L01 transmitter/receiver controller for smooth multi-DOF motion. MicroPython firmware enables calibration, interpolation, safety, and real-time teleoperation.
